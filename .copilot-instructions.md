# Copilot Instructions

## Running commands

- Prefer using `make` targets when the project has a `Makefile`.
- Before running tests or linters, read the project's `README` (or `README.md`) to find the correct commands.

## Docker

- If you receive Docker errors when running tests (e.g. port conflicts, container issues), kill existing containers first:
  ```bash
  docker kill $(docker ps -q)
  ```
  Then retry the command.
